# lua-rtmidi

Lua bindings for [RtMidi](http://www.music.mcgill.ca/~gary/rtmidi/index.html).

[API documentation](./doc/index.md)

# Building

**lua-rtmidi** requires CMake 3.12 or above. Only tested on Windows.

```sh
take build
cmake ..
cmake --build . --target lua-rtmidilove
```
